# Home Chef Meals

A simple React app where you can browse meals made by home chefs, filter them, and build your meal selection.

## What it does

Shows a list of meals with name, price, and availability. You can add available meals to your selection and see the total price. There's also a toggle to view all meals (including unavailable ones) and a sort button for price.

## Features

- Shows available meals by default
- Toggle to view all meals
- Add meals to selection (no duplicates, unavailable meals can't be added)
- Total price of selected meals
- Sort meal list by price: Low → High / High → Low

## How to run

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build for production

```bash
npm run build
```
